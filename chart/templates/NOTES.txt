Application Configuration:
{{- range $key, $value := .Values.env }}
  - {{ $key }}: {{ $value | default "[not set]" }}
{{- end }}

To override environment variables when installing the chart:

  helm install blockchain-ops ./chart -n ns-interview \
    --set env.NODE_URL="your-node-url" \
    --set env.USER_ADDRESS="your-address" \
    --set env.START_BLOCK="your-start-block" \
    --set env.NB_BLOCKS="number-of-blocks"

Or use a values file with your configuration:

values.yaml:
  env:
    NODE_URL: "your-node-url"
    USER_ADDRESS: "your-address"
    START_BLOCK: "your-start-block"
    NB_BLOCKS: "10"
